Типы атрибутов:
1. boolean. Поля - value(true/false), value_true, value_false
2. string. Поля - value
3. array. Поля - begin_symbol, split_symbol, end_symbol, values(split by comma)
4. object. Поля - link to an object

Необходимо тегировать части команды и обрабатывть отдельно, определяя необходимые объекты для текущего задания.
Для каждого задания существует свой список объектов.
CREATE - объект, [аттрибуты объекта], [местоположение]
DELETE - объект, идентификатор (имя, местоположение, местоимение)

объект можно обнаружить, сравнивая слова со всеми звуками, определёнными в шаблоне объекта
аттрибуты можно обнаружить, сравнивая слова со всеми звуками, определёнными в шаблоне аттрибута
местоположение можно обнаружить, сравнивая слова со всеми ключевыми словами, определяющими местоположение

-Пример глобального тегирования:

-Создать статический класс potato после класса vegetable
-Добавить родителя к классу potato значение vegetable

	SINGLE_CMD	- ACTION

	BINARY ATTR	- [ACTION, default CHANGE] OBJ_TYPE_NAME [NAME, default neartest object in CONTEXT] to(на) [not][ ]BINARY_ATTR_NAME
	STRING_ATTR	- [ACTION, default CHANGE] STRING_ATTR_NAME TYPE_NAME [NAME, default neartest object in CONTEXT] to(на) STRING_ATTR_VALUE
